# Household Poverty Classification using Machine Learning Techniques (Case in Ghana)

## Overview
This project develops machine learning models to classify household poverty status using household survey data.

## Objective
The goal is to identify poor households using socio-economic and household-level indicators.

## Methods
- Logistic Regression Model
- Linear Discriminant Analysis
- K-Nearest Neighbor
- Decision Tree
- Naive Bayes
- Random Forest 
- Gradient Boosting
- Support Vector Machine (SVM)

## Workflow
- Data cleaning
- Feature engineering
- Model development
- Cross-validation
- Performance evaluation

## Results
The models achieved approximately 90% classification accuracy, with strong performance from ensemble methods.

## Tools
- R
- caret
- randomForest
- e1071
- MASS
- ggplot2

## Note
This project was completed during an internship at **ICF International Inc.**, Reston, VA, USA, in 2024. The original survey dataset used in this project is confidential and cannot be publicly shared.
